## Script Mapping

- scripts/download_enemy_assets_openmoji.ps1: OpenMoji enemy replacement mapping
- scripts/download_ui_emoji_openmoji.ps1: OpenMoji UI status mapping
- scripts/download_beautify_assets.ps1: OpenMoji + Mixkit batch download
- scripts/download_beautify.js: OpenMoji + Mixkit batch download
- scripts/dl_sfx.js: Mixkit SFX downloader
- scripts/dl_bgm.js: Pixabay BGM fallback downloader

## Notes

- The project supports Windows local font fallback if packaged font loading fails.
- Audio runtime prefers MP3 BGM files. If an MP3 is missing, the code falls back to same-name WAV for compatibility.
- Keep this file synchronized whenever asset sources or filenames change.
