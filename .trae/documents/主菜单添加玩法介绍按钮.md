# 主菜单添加玩法介绍按钮

## 任务概述
在游戏主菜单界面添加"玩法介绍"按钮，玩家点击后可查看游戏规则。按钮大小与"开始游戏"按钮相同，位于"开始游戏"和"退出游戏"按钮中间，三个按钮不重叠。

## 当前布局分析

### 现有按钮位置
- `menuStartRect{470.f, 430.f, 340.f, 82.f}` - 开始游戏按钮
- `menuExitRect{470.f, 540.f, 340.f, 82.f}` - 退出游戏按钮

### 新布局设计
三个按钮垂直排列，间距50像素：
- 开始游戏: Y=430
- 玩法介绍: Y=480 (新增)
- 退出游戏: Y=530

修改后坐标：
```cpp
sf::FloatRect menuStartRect{470.f, 380.f, 340.f, 82.f};   // 开始游戏
sf::FloatRect menuRulesRect{470.f, 480.f, 340.f, 82.f};   // 玩法介绍 (新增)
sf::FloatRect menuExitRect{470.f, 580.f, 340.f, 82.f};    // 退出游戏
```

## 实现步骤

### 1. 添加新的游戏阶段枚举
- 在 `Phase` 枚举中添加 `Rules` 阶段

### 2. 添加按钮区域定义
- 添加 `menuRulesRect` 成员变量
- 修改 `menuStartRect` 和 `menuExitRect` 的Y坐标

### 3. 添加关闭按钮区域
- 添加 `rulesCloseRect` 用于关闭规则页面

### 4. 修改handleEvent函数
- 在主菜单阶段处理"玩法介绍"按钮点击
- 在 `Rules` 阶段处理关闭按钮点击

### 5. 修改render函数
- 添加 `Rules` 阶段的渲染分支
- 在 `Rules` 阶段使用主菜单BGM

### 6. 添加renderRules函数
- 渲染游戏规则内容
- 包含关闭按钮

### 7. 修改renderMainMenu函数
- 渲染"玩法介绍"按钮

## 游戏规则内容

包含以下内容：
- 游戏目标
- 卡牌类型说明
- 战斗流程
- 资源系统简介
- 快捷键说明
