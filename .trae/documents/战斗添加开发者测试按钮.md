# 战斗中添加"开发者测试"按钮计划

## 任务概述
在每场战斗中新增"开发者测试"按钮，点击后对敌人造成999伤害。按钮位于BGM音量按钮旁边，尺寸和形式与其一致。

## 当前BGM按钮分析

### BGM按钮定义
- 位置：`bgmToggleRect{1220.f, 680.f, 40.f, 20.f}`
- 尺寸：40x20像素
- 位置：右下角

### 新按钮位置设计
- 开发者测试按钮放在BGM按钮左侧
- 位置：`devTestRect{1170.f, 680.f, 40.f, 20.f}`
- 两个按钮之间间距10像素

## 实施步骤

### 1. 添加按钮矩形定义
在成员变量区域添加：
```cpp
sf::FloatRect devTestRect{1170.f, 680.f, 40.f, 20.f};
```

### 2. 添加renderDevTest函数
创建渲染函数，与renderBgmToggle风格一致：
- 绘制按钮背景
- 显示"测试"文字

### 3. 修改handleEvent函数
在Battle阶段添加点击处理：
```cpp
if (devTestRect.contains(mouse)) {
    playSfx(sfxUiClickBuf, 50.f);
    dealDamage(enemy, 999, false);
    pushLog("开发者测试：造成999点伤害");
    return;
}
```

### 4. 修改renderBattle函数
在战斗界面渲染开发者测试按钮：
```cpp
renderDevTest();
```

### 5. 修改render函数
确保在Battle阶段渲染开发者测试按钮。
